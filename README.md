## 🇮🇷 درباره پروژه
**DoOnTime** یک سامانه مدیریت یادآوری مدرن است که به شما کمک می‌کند هیچ‌کدام از وظایف، سررسیدها یا قرارهای خود را فراموش نکنید. این سیستم با رابط کاربری گلس‌مورفیسم (Glassmorphism) و پشتیبانی از تقویم جلالی، تجربه‌ای لذت‌بخش از مدیریت زمان را ارائه می‌دهد.

### ویژگی‌های کلیدی:
* **یادآوری چندکاناله:** ارسال هشدار از طریق تلگرام، پیامک (IPPanel) و ایمیل.
* **سیستم مالی هوشمند:** نمایش موجودی پنل به تومان با تبدیل خودکار و مدیریت تراکنش‌ها.
* **داشبورد مدرن:** طراحی ریسپانسیو با پشتیبانی از Dark Mode.
* **مدیریت زمان:** تبدیل دقیق تاریخ میلادی به شمسی و تنظیم یادآورهای تکرار شونده.

## 🇬🇧 About Project
**DoOnTime** is a modern, multi-channel reminder management system designed to ensure you never miss a task or deadline. With a stunning Glassmorphism UI and Jalali calendar support, it provides a seamless time management experience.

### Key Features:
* **Multi-Channel Notifications:** Alerts via Telegram, SMS (IPPanel), and Email.
* **Financial Integration:** Account balance monitoring and secure payment gateway integration.
* **Modern Dashboard:** Responsive design with full Dark Mode support.
* **Time Management:** Precise Jalali calendar conversion and recurring reminders.

---

## 🚀 تکنولوژی‌ها / Technologies
* **Backend:** PHP 8+
* **Frontend:** Bootstrap 5, FontAwesome
* **Database:** MySQL
* **APIs:** IPPanel SMS API, Telegram Bot API
* **Libraries:** Carbon (for date handling)
