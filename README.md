# ⏳ DoOnTime
### سیستم هوشمند یادآوری چندکاناله (SMS, Telegram, Email)

[**مشاهده دموی زنده**](https://doontime.mohandeseit.com/)

## 🇮🇷 معرفی پروژه
**DoOnTime** یک سامانه مدیریت یادآوری مدرن است که با هدف سازماندهی دقیق امور روزمره، مالی و کاری طراحی شده است. این سیستم با استفاده از تکنولوژی‌های روز وب، تجربه‌ای سریع و بدون دغدغه از مدیریت زمان را ارائه می‌دهد.

---

## 🎯 کاربردهای DoOnTime
| کاربرد | شرح مختصر |
| :--- | :--- |
| **پرداخت قسط** | جلوگیری از جریمه دیرکرد وام‌های بانکی |
| **سررسید چک** | حفظ اعتبار و مدیریت دقیق حساب‌ها |
| **پرداخت قبوض** | پرداخت به موقع قبض آب، برق و گاز |
| **مصرف دارو** | هشدارهای دقیق برای نوبت‌های پزشکی |
| **تاریخ تولد** | تبریک به موقع تولد و سالگرد ازدواج |
| **مدیریت کارهای روزانه** | بهترین جایگزین برای تو-دو لیست‌های کاغذی |

---

## ⚡ امکانات هوشمند سامانه
*   **نوتیفیکیشن تلگرام:** دریافت هشدارهای لحظه‌ای مستقیماً در چت شخصی تلگرام بدون نیاز به نصب اپلیکیشن اضافه.
*   **پیامک (SMS):** حتی بدون دسترسی به اینترنت، یادآورهای ضروری مستقیماً به گوشی شما پیامک می‌شوند.
*   **اطلاع‌رسانی ایمیلی:** ارسال گزارش‌ها و یادآورهای رسمی به صورت قالب‌بندی شده برای خود یا تیم.
*   **دوره‌های تکرار شونده:** تنظیم کارهای روتین با بازه‌های زمانی منعطف (روزانه، هفتگی، ماهانه یا سالانه) تنها با چند کلیک.

## 🛠 تکنولوژی‌ها
*   **Backend:** PHP 8+
*   **Frontend:** Bootstrap 5, Glassmorphism UI
*   **Database:** MySQL
*   **APIs:** IPPanel SMS, Telegram Bot API
*   **Libraries:** Carbon / Jalalian
